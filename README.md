# softka_performance
prueba de performance
Este archivo contiene instrucciones de como realizar la prueba 

Instrucciones
* Tener descargado Jmeter
* Abrir Jmeter
* Descargar el "proyecto" de performance
* Abrir el achivo hilos Softka.jmx
* El archivo LoginList.csv ubicarlo en una ruta a su criterio
* Una vez cargado el proyecto en Jmeter, ir a la opcion "Configuración del CSV Data Set"
  * En el filename / Nombre del archivo selecciones la ruta donde guardo LoginList.csv
  * dentro de Nombres de variables ingresar los siguiente [user,passwd]
  * El delimitador es ","    
* Considerando que los demas datos y configuracion estan guardados en el archivo solo falta en ejecutar el boton Play
